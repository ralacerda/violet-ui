## Icon

You can only use Material Symbol icons. The string prop should not include the 'material-symbols:' part.

So use `done` instead of `material-symbols:done`.

We are using Iconify Vue2 component. 
You can find a list of all Material Symbol icons on [iconify page](https://icon-sets.iconify.design/material-symbols/).
